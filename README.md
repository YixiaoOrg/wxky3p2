#### 班级简介
本班由无锡科技职业技术学院（3年）与盐城师范学院（2年）分段培养
（在无锡科技职业学院学习三年考核合格者可转入分段培养本科院校学习两年，毕业后颁发分段培养院校本科文凭.）
